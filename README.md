# Онлайн-сервіс для вивчення англійської

## Опис проєкту
Це веб-застосунок для вивчення англійської мови з генерацією завдань на основі тематики.  
Користувач може обрати тему, отримати завдання та перевірити свої відповіді.

## Мета
Набути практичних навичок створення MVP веб-додатку з базовим функціоналом та тестуванням користувацьких сценаріїв.

## Використані технології
- HTML5
- CSS3
- JavaScript
- Git та GitHub для контролю версій

## Функціонал MVP
1. Вибір теми завдань.
2. Генерація завдань за обраною темою.
3. Перевірка правильності відповідей.
4. Показ результату користувачеві.

## Інструкція запуску
1. Склонуйте або завантажте репозиторій на свій комп'ютер.
2. Відкрийте файл `index.html` у будь-якому браузері.
3. Оберіть тему та натисніть "Згенерувати завдання".
4. Введіть відповіді у поля та натисніть "Перевірити відповіді" для отримання результату.
